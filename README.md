# Lift-Simulation
Aim is to create an ember app where you can simulate lift mechanics.

## Installation
```shell
$ git clone git@github.com:chanchalupadhyay/lift-simulation-emberjs.git
$ cd Lift-Simulation-emberjs
$ npm install
$ ember serve
```

### Features to be executed: 

- [x] simulation engine
